Getting Started

Quick Start

A full 3D scene in under 15 lines.

Installation

npm i bunny-3d

Minimal scene

import Bunny from './bunny-3d/bunny/bunny.js';

const app = new Bunny();
app.pCamera();       // perspective camera
app.AddToDom();      // mount canvas
app.addControls();   // orbit controls

app.light('AmbientLight', 0xffffff, 0.6);
app.light('DirectionalLight', 0xffffff, 1);

const box = app.sCube(1, 1, 1);

function animate() {
  requestAnimationFrame(animate);
  box.rotation.y += 0.01;
  app.controls.update();
  app.renderFrame();
}
animate();
Core

Cameras

Choose a camera type before calling renderFrame(). All cameras are added to the scene and positioned at z = 5 by default.

pCamera ( fov, aspect, near, far ) method
Creates a PerspectiveCamera ‚Äî the standard camera for 3D scenes. Mimics how the human eye perceives depth.
ParameterTypeDefaultDescription
fovnumber75Vertical field of view in degrees
aspectnumberinnerWidth / innerHeightCanvas aspect ratio
nearnumber1Near clipping plane distance
farnumber1000Far clipping plane distance
oCamera ( left, right, top, bottom, near, far ) method
Creates an OrthographicCamera. No perspective distortion ‚Äî great for 2D scenes, UI overlays, and isometric views.
ParameterTypeDefaultDescription
leftnumber-innerWidth/2Left frustum plane
rightnumberinnerWidth/2Right frustum plane
topnumberinnerHeight/2Top frustum plane
bottomnumber-innerHeight/2Bottom frustum plane
cubeCamera ( near, far, resolution ) method
Creates a CubeCamera for real-time environment mapping and reflections. Resolution defaults to 256.
arrayCamera ( cameras[] ) method
Creates an ArrayCamera for multi-viewport rendering. Pass an array of PerspectiveCameras.
stereoCamera () method
Creates a StereoCamera for VR and anaglyphic 3D effects. Does not add to scene automatically.

Renderer

Mounts the WebGL canvas and drives the render loop.

AddToDom () method
Appends the WebGL canvas to document.body, sets pixel ratio (capped at 2), and attaches a resize listener that keeps the canvas and camera in sync.
Note: Call this once after creating your camera. The resize handler is included ‚Äî you don't need to call handleResize() separately.
renderFrame () method
Renders one frame. Throws if no camera has been initialized. Call inside a requestAnimationFrame loop.
function animate() {
  requestAnimationFrame(animate);
  app.renderFrame();
}
animate();

Controls

addControls ( enableDamping ) method
Attaches OrbitControls to the active camera, enabling mouse/touch pan, zoom, and orbit. Requires a camera to exist first.
ParameterTypeDefaultDescription
enableDampingbooleantrueSmooth inertia on orbit movement
Important: When damping is enabled, call app.controls.update() inside your animation loop or controls won't work correctly.

sCube ( w, h, d, segX, segY, segZ, color, map ) ‚Üí Mesh
Creates a box. First three params are width, height, depth. Segment params control subdivision density.
const box = app.sCube(2, 1, 1, 1, 1, 1, 0x3a7bc8);
box.position.set(0, 0.5, 0);
sPlane ( w, h, segW, segH, color, map ) ‚Üí Mesh
Creates a flat plane. Rotate it to use as a floor.
const floor = app.sPlane(20, 20, 1, 1, 0x444444);
floor.rotation.x = -Math.PI / 2;
sSphere ( radius, widthSeg, heightSeg, color, map ) ‚Üí Mesh
Creates a sphere. Higher segment counts produce smoother geometry.
sCapsule ( radius, length, capSeg, radSeg, color, map ) ‚Üí Mesh
Creates a capsule (cylinder with rounded caps). Useful for character colliders and stylized shapes.
sCylinder ( rTop, rBot, h, radSeg, hSeg, color, map ) ‚Üí Mesh
Creates a cylinder. Set rTop and rBot to different values for a tapered shape.
sCone ( radius, h, radSeg, hSeg, color, map ) ‚Üí Mesh
Creates a cone shape.
sTorus ( radius, tube, radSeg, tubeSeg, color, map ) ‚Üí Mesh
Creates a donut/torus shape. radius is the ring radius, tube is the tube thickness.

Lighting

All lights are added to the scene and returned for further configuration like positioning.

light ( type, color, intensity, distance, angle, penumbra, decay ) ‚Üí Light
Creates and adds a light by type string. Throws if an unsupported type is passed.
ParameterTypeDefaultDescription
typestring‚ÄîLight type string (see below)

intensitynumber1Light brightness
distancenumber0Max range (PointLight, SpotLight)
anglenumberœÄ/3Cone angle in radians (SpotLight)

AmbientLight
Uniform fill light. No direction, no shadows. Good base layer.
DirectionalLight
Parallel rays from infinite distance. Great for sunlight.
HemisphereLight
Sky/ground gradient. Natural outdoor ambience.
PointLight
Omnidirectional point source. Like a light bulb.
SpotLight
Cone-shaped beam with angle and soft penumbra edge.
RectAreaLight
Rectangular area light. Default 10√ó10 size.
AmbientLightProbe
IBL ambient contribution from a light probe.
HemisphereLightProbe
Hemisphere-based light probe for natural fills.
// Typical setup
app.light('AmbientLight', 0xffffff, 0.5);
const sun = app.light('DirectionalLight', 0xfffae0, 1.5);
sun.position.set(5, 10, 5);
sun.castShadow = true;
Loaders

Asset Loaders

Three async methods for importing 3D and vector assets directly into your scene.

gltf ( url ) async
Loads a .gltf or .glb model via GLTFLoader. Adds the model's scene to the Bunny scene and returns the full gltf object. Errors are caught and logged.
const model = await app.gltf('./models/character.glb');
model.scene.position.set(0, -1, 0);
model.scene.scale.setScalar(0.5);
obj ( url ) async
Loads a .obj model via OBJLoader. The object is added to the scene automatically. Note: OBJ does not include materials ‚Äî pair with MTLLoader manually if needed.
await app.obj('./models/mesh.obj');
svg ( url ) async
Loads an .svg file via SVGLoader and extrudes each path into 3D geometry using ShapeGeometry. Colors from the SVG are preserved. All shapes are grouped and added to the scene.
Note: SVG shapes are flat (no depth). Rotate or reposition the returned group to place it in your scene.
await app.svg('./logo.svg');
